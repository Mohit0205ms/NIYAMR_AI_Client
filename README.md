## Getting Started

First, run the developement server:

```bash
npm install
# or
yarn install
```

Second, run the developement server:

```bash
npm run dev
# or
yarn dev

```

Open (http://localhost:3000) http://localhost:3000 with your browser to see the resulat.

You can start edit the page by modifying app/page.tsx. The page auto update as you edit the file.

This project uses (next/font)https://nextjs.org/docs/app/building-your-application/optimizing/fonts to automaticly optimize and load (Geist)https://vercel.com/font, a new font familly for Vercel.
